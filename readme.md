# 💼 Personal Portfolio Website

A clean, responsive portfolio website built using modern frontend technologies to showcase projects, skills, experience, and certifications.

---

## 🚀 Overview

This portfolio is designed to present a structured and minimal interface highlighting:

- Professional experience  
- Technical skills and tools  
- Academic coursework  
- Projects with GitHub links  
- Certifications  

The design focuses on readability, simplicity, and strong visual hierarchy.

---

## 🛠 Tech Stack

**Frontend**
- React.js
- Vite
- JavaScript (ES6+)
- CSS (Custom styling)

**Styling**
- CSS Variables
- Flexbox & Grid Layout
- Responsive Design

---

## ✨ Features

- 📄 Multi-section layout (About, Skills, Projects, Experience, Certifications)
- 🎯 Clean typography and structured UI
- 🧩 Reusable components (ProjectCard, Sections)
- 🔗 GitHub project integration
- 📱 Fully responsive design
- 🖼 Profile image support
- ⚡ Fast performance with Vite
